# 🏛️ Barangay Management System

A web-based Barangay Management System developed for Barangay Bacolod, Tabaco City. The system streamlines barangay operations by digitizing resident records, managing barangay officials, and providing secure role-based access control for different users.

Built using **PHP 8.2.12**, **Laravel 12.62.0**, **Vite 7.3.5**, **Tailwind CSS 4.3.1**, **HTML5**, **CSS3**, and **JavaScript**, the application provides a modern, responsive, and user-friendly platform for residents and barangay personnel.

## 🚀 Key Features

### 👤 User Registration

* Resident Registration
* Barangay Official Registration
* Secure user authentication and login
* Account management and profile updates

### 🔐 Role-Based Access Control

The system supports three user roles:

#### Resident

* View personal information
* Access barangay services
* Monitor submitted requests

#### Barangay Official

* Manage resident records
* Monitor community data
* Process resident requests and applications

#### Administrator

* Full system management
* User and role administration
* System monitoring and maintenance
* Generate reports and analytics
